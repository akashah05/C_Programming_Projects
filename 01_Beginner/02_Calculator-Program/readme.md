# Number Guessing Game (Beginner)
A simple C program that performs basic arithmetic operations.
The user selects an operation (+, -, *, /) and enters two numbers.
The program then displays the result. The user can exit anytime by pressing e.

## Features
    Addition
    Subtraction
    Multiplication
    Division
    Exit option
    Basic input validation

## Concepts Used
    Functions   
    Conditional statements (if / else)
    Loops (while)
    User input with scanf
