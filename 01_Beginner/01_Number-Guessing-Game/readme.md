# Number Guessing Game (C)

A beginner-friendly C program where the user guesses a randomly generated number.

## Features
- Random number generation
- Limited number of attempts
- Hint system (Too High / Too Low / Near)
- Input validation
- Replay option

## Concepts Used
- Variables and constants
- Conditional statements
- Loops
- Functions
- Standard library functions (`rand`, `srand`, `time`)

## Version History
- v1.0 – Basic guessing logic with a fixed number
- v2.0/2.1 – Random numbers, attempts limit, hints, replay, input validation
